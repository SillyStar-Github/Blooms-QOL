# Blooms QOL
A repository that hosts the source code for all the QOL mods included in the [Blooms translation](https://github.com/Teyliu/PVZF-Translation).

# Instructions
Adjust the ``GamePath`` variable in ``Blooms QOL.csproj`` to the folder that contains your PvZ: Fusion installation. Then, you should be able to build the mod and it will copy to your ``Mods`` folder automatically.
